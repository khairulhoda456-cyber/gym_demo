#Prompt which is used to develop a website

Design Requirements:
- Dark theme (black + neon green / cyan accent)
- Glassmorphism UI (blurred cards, transparent backgrounds)
- Smooth animations and hover effects
- Fully responsive (mobile + desktop)
- Clean, modern typography (use Google Fonts like Poppins)

Sections to include:
1. Fixed Navbar with logo and smooth scroll links
2. Hero Section:
   - Full-screen background image with overlay
   - Big bold heading (e.g., "Build Your Dream Body")
   - Subheading
   - Strong CTA button ("Join Now")
   - Animated entrance (fade/slide)

3. Urgency Banner:
   - Limited-time offer (e.g., "50% OFF Today Only")

4. Stats Section:
   - Show numbers like 5000+ Members, 50+ Trainers, 10+ Years Experience
   - Animated counters if possible

5. Programs Section:
   - Cards for Weight Training, Fat Loss, Yoga
   - Hover effects and icons

6. Pricing Section:
   - 3 pricing cards (Basic, Pro, Elite)
   - Highlight middle plan
   - CTA buttons

7. Testimonials:
   - At least 1–2 customer reviews

8. Contact / CTA Section:
   - Join Now button or simple form

9. Footer:
   - Minimal, clean

Functionality:
- Smooth scrolling
- Button click shows alert (Join Now)
- Simple animations using CSS or JavaScript
- Optional: scroll reveal animations

Constraints:
- Everything must be inside ONE index.html file
- Use internal <style> and <script>
- Use free image URLs (Unsplash or similar)
- Code should be clean, well-structured, and commented

Goal:
The website should look high-end, attractive, and designed to convert visitors into gym customers.
